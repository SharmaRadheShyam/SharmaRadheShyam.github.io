Domain Adaptationa in Computer Vision

Phd Research Work

Deep convolution neural network architectures have brought impressive success in computer vision. However, a large amount of labeled training data is needed for improved performance. In practice, the quantity of available data is both limited and unlabeled. Although it might be possible to find a dataset big enough for training large scale deep models, the supervised data available (source domain) could be from a domain different from the application domain (target domain). We would need to transform or adapt the classifiers to work on the data from the target domain. Adapting a discriminative classifier or predictor due to presence of a shift between training and test distributions is known as domain adaptation (DA).

Gaussioan Process based Halluciantion of features for object classification

Term Project-April 2015

Learn a classifier using Gaussian Process regression model to hallucinate the unseen features to the known features. Assign the label for each object with corresponding feature label. Bootstrapping improves the correct recognition ratio performance. [PPT]

GPS based Vehicle Tracking System

B.E. Project April- 2012

This Project presents an automotive localization system using GPS and GSM-SMS services. GPS receiver gets the location information from satellites in form of latitude and longitude. The Microcontroller processes this information and this processed information is sent to the user/owner using GSM modem. This project is for automobile position and status, very useful in case of car theft situations, for monitoring adolescent drivers by their parents as well as in car tracking system applications.
