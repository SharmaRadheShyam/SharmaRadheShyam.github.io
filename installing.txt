Step to install OpenCV on Windows Plateform:-

1. Double Click on OpenCV application file and type the extract path "C:/opencv2411" and extract it .
2.Cut all the folder fron C:/opencv2411/opencv and paste them to C:/opencv2411, delete opencv folder.
3.Go to computer and right click then select properties, Left hand side click on "advance system setting."

4.Go to advanced and click on "environmental varriables", There are two environmenatal varriables : user varriables and system varriables.


5.In the user varrible part click on new. write variable name :- OPENCV_BUILD and variable value:- C:\opencv2411\build.
6. Now in the system variable select path and click on edit , in the variable value after the ";" semicoloumn ,write :- C:\opencv2411\build\x64\vc12\bin;
